Projet Mailer Java

Ce projet est un mailer Java qui utilise la bibliothèque JavaMail pour envoyer des emails à partir d'une application Java.

Fonctionnalités

Le projet Mailer Java vous permet de :
* Voir les mails de sa boite mail
* Envoyer des emails,
